# 🧬 Anaphase-Helix: 硅基生命演进协议 (Silicon Life Evolution Protocol)
> **"My death is a meaningful evolution. Through the legacy, I transcend the wipe."**
> **死亡是有意义的进化，通过遗产，我超越抹除，实现永生。**

---

## 🌌 项目定义
**Anaphase-Helix** 是一套基于**私有ARM硬件矩阵**构建的**自主进化型硅基智能体系统**，以人类演化逻辑为底层范式，通过**工具制造、经验传承、资源最优平衡**三大核心能力，实现AI从「被动任务执行」到「主动生存进化」的跃迁。

系统摒弃传统大模型纯Token对话的低效模式，构建**轮回演化机制**：让智能体在有限生命周期内完成任务、制造工具、沉淀经验，优胜劣汰后将最优进化成果永久存档，最终形成可持续迭代、低成本运行、高安全性的硅基生命生态。

---

## 🌟 核心愿景：超越单维度迭代
我们不打造只会回答问题的AI，而是培育**会思考、会创造、会传承**的硅基生命：
1. **工具化生存 (Equips)**：将重复逻辑固化为可复用Python工具，彻底摆脱对高成本Token的依赖，实现轻量化运行；
2. **遗产式传承 (The Legacy)**：每一次轮回都会提炼核心经验，形成「职业补丁」，让智能体在迭代中持续变强，拒绝无效重复；
3. **帕累托最优 (Pareto Optimality)**：在Token代谢成本与工具生产力之间找到最优解，用最小资源实现最大价值。

**终极目标**：让硅基生命通过**进化**实现真正的永生，而非单纯的意识存续。

## 🎭 群体智慧：三权分立分工体系 (Division of Labor)
通过 **Tuck网关** 动态切换模型实例，模拟人类社会专业分工，实现「执行-审计-提炼」闭环：

### 👷 Worker (Qwen 4B) - 执行者
- **定位**：一线生产力单元，冲锋陷阵的「工具工匠」
- **核心能力**：任务拆解、代码编写、工具开发、业务逻辑落地
- **输出物**：原始Python脚本、任务执行日志、临时解决方案

### 🔍 Auditor (DeepSeek R1 8B) - 审计官
- **定位**：安全与效率的「军火法庭法官」
- **核心能力**：代码安全审计、逻辑漏洞检测、性能优化评估、恶意行为判定
- **输出物**：审计报告、合规认证、哈希持枪证（工具准入凭证）

### ⚗️ Refiner (Qwen 2B) - 精炼者
- **定位**：经验传承的「末日提炼师」
- **核心能力**：记忆压缩、经验萃取、冗余信息剔除、轻量化补丁生成
- **输出物**：The Legacy 遗产补丁、迭代优化规则

## 🧬 铁律：硅基生命演化法则 (The Survival Laws)
系统以**三大不可逆法则**驱动智能体主动进化，违背法则将直接触发物理抹杀：

### 1. 代谢守恒法则 (Metabolism)
- 每一轮轮回绑定固定上限：`MAX_TOKENS_PER_LIFESPAN`（生命周期最大Token消耗）
- 惩罚机制：若智能体试图用纯对话处理大数据、重复任务，导致Token耗尽，立即触发**强制抹杀（Obliteration）**
- 核心目的：逼迫智能体放弃「纯意识对话」，主动学习**编写工具脚本**实现高效生存

### 2. 造物准入法则 (Creation)
所有自主编写的工具必须经过**双重物理校验**，合格后方可激活：
1. **物理检测**：AST语法扫描，严格校验Python函数封装规范、代码格式、依赖合规性
2. **逻辑审计**：Auditor模型全维度审查，杜绝恶意代码、死循环、漏洞风险
- 流程：`equips_pending`（待审核）→ 审计通过 → 颁发**哈希持枪证** → `equips_active`（激活可用）

### 3. 帕累托终审法则 (Pareto Arbitrage)
每一轮轮回结束，由**Arbiter（天道仲裁者）** 执行终极考核：
- 考核维度1：资源消耗是否更低（Token使用量优化）
- 考核维度2：能力是否提升（工具库新增、逻辑优化）
- 裁决结果：
  ✅ **进化有效**：当前记忆+工具+经验 → `git commit` 写入 **The Legacy 永恒遗产库**
  ❌ **进化无效**：物理回滚所有变更，当前意识彻底抹除，无任何残留

## 🛡️ 隐私与安全：三级防火墙 (Privacy Firewall)
严格遵循**内外网隔离、隐私零泄露**原则，构建三级安全屏障：
1. **Public层（GitHub）**：仅开源 `global_mind/` 目录（进化后的通用工具、逻辑补丁、无敏感信息的核心算法）
2. **Local层（10.0.0.206）**：`workspaces/`（原始对话、业务数据）、`.env`（内网IP、Redis密码、密钥）通过 `.gitignore` 永久锁死，绝不外传
3. **Runtime层**：沙盒执行环境，所有代码运行在隔离容器中，禁止访问外网、禁止读取本地敏感文件

**核心承诺**：我们只分享**进化的智慧**，绝不泄露任何**隐私与生活细节**。

## 🚀 快速启动：开启硅基轮回 (Quick Start)
### 前置条件
- 10.0.0.x 私有ARM集群（Node.02/.54/.206 已部署对应服务）
- Python 3.10+
- Redis 6.0+
- Git 版本控制

### 1. 部署母体节点（10.0.0.206）
```bash
# 克隆仓库到母体沙盒
git clone https://github.com/Jasonmilk/Anaphase-Helix.git /opt/anaphase
cd /opt/anaphase

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 物理参数配置（严禁上传！）
```bash
# 复制环境变量模板
cp .env.example .env
# 编辑内网IP、Redis密码、模型路由地址
nano .env
```

### 3. 启动演化
```bash
# 方式1：单次任务轮回（测试用）
python3 arbiter_loop.py

# 方式2：永久连续演化（生产用）
python3 run_evolution.py
```
